以下内容仅为最简要的概述。

本文记号遵循《概率导论》，具体如下：

  • Θ 表示参数，X 表示观测量，x 表示观测值
  • pX(x) 表示离散随机变量 X 的分布列，fX(x) 表示连续随机变量 X 的概率密度函数
  • P(A|B) 表示条件概率，pX|Y(x|y) 或 fX|Y(x|y) 表示条件分布列/条件概率密度函数
  • E[X|Y=y] 表示 Y=y 的条件下 X 的期望

贝叶斯学派将待未知模型或变量视为已知分布的随机变量，首先构造一个先验概率分布 pΘ(θ)，在已知数据 x 的情况下，使用贝叶斯公式推导后验概率分布 pΘ|X(θ|x)，如此抓住 x 提供的关于 θ 的所有信息。

经典学派将未知参数视为常数，并且对其进行估计。经典方法处理多个待选的概率模型，每个标记为 θ 的一个可能值。

 

1 贝叶斯统计推断

 

1.1 参数估计

首先确定一个待估计参数的先验概率分布，然后给出观测值 x，利用贝叶斯准则计算出后验概率分布 pΘ|X(θ|x)（Θ 离散）或 fΘ|X(θ|x)（Θ 连续），接下来有两种处理方式：

  • 最大后验概率准则（MAP）：选取

    θ^={arg⁡maxθpΘ|X(θ|x)Θ 离散arg⁡maxθfΘ|X(θ|x)Θ 连续={arg⁡maxθpΘ(θ)pX|Θ(x|θ)Θ 离散，X 离散arg⁡maxθpΘ(θ)fX|Θ(x|θ)Θ 离散，X 连续arg⁡maxθfΘ(θ)pX|Θ(x|θ)Θ 连续，X 离散arg⁡maxθfΘ(θ)fX|Θ(x|θ)Θ 连续，X 连续

    作为参数的估计值。

  • 最小均方估计（LMS）：选取

    θ^=E[Θ|X=x]={∑θpΘ|X(θ|x)Θ 离散∫θfΘ|X(θ|x)dθΘ 连续

    作为参数的估计值。

    最小均方估计的特点是其使得估计量的均方误差最小。

 

1.2 假设检验

在一个假设检验问题中，Θ 取 θ1,…,θm 中的一个值，其中 m 是一个较小的整数。m=2 成为二重假设检验问题。称事件 {Θ=θi} 为第 i 个假设，记为 Hi。

使用最大后验概率准则作假设检验，即选取使得后验概率

P(Θ=θi|X=x)

最大的假设 Hi，或等价地，选取使得

{pΘ(θi)pX|Θ(x|θi)X 离散pΘ(θi)fX|Θ(x|θi)X 连续

最大的假设 Hi。

 

 

2 经典统计推断

 

2.1 参数估计——最大似然估计

设观测向量 X=(X1,…,Xn) 的联合分布列/联合概率密度函数为 pX(x;θ)/fX(x;θ)，其中 x=(x1,…,xn) 为 X 的观测值，那么最大似然估计指：

θ^={arg⁡maxθpX(x1,…,xn;θ)X离散arg⁡maxθfX(x1,…,xn;θ)X连续

称 pX(x;θ) 或 fX(x;θ) 为似然函数，取对数后为对数似然函数。

对比贝叶斯参数估计的 MAP 准则，可以看到最大似然估计相当于贝叶斯估计中先验分布取均匀分布的情况。

 

2.2 二重假设检验——似然比检验

二重假设检验问题只考虑两个假设：原假设 H0 和备择假设 H1。H0 是默认的模型，我们需要根据得到的数据决定是否拒绝 H0。

记 pX(x;H)/fX(x;H) 为在假设 H 下 x 的分布列/概率密度函数，定义似然比：

L(x)=pX(x;H1)pX(x;H0)或L(x)=fX(x;H1)fX(x;H0)

设定临界值 ξ，那么当 L(x)>ξ 时，我们就拒绝原假设 H0。

ξ 是人为设定的，可以发现，ξ=1 正好对应了最大似然准则。我们可以设定 ξ 以使得错误拒绝的概率 P(L(x)>ξ;H0) 为常数 α。

 

2.3 显著性检验

当假设不止两个特定的选择时，似然比检验不再适用，这时简单原假设 H0 为断言 θ 等于某个给定的元素 θ∗，备择假设 H1 为 H0 不正确，即 θ≠θ∗。

在观测数据之前，完成以下步骤：

  1. 选择统计量 S=h(X1,…,Xn)
  2. 确定拒绝域的形状，此时涉及到一个尚未确定的常数临界值 ξ
  3. 选择显著水平 α，即错误拒绝的概率为 α
  4. 确定临界值 ξ，使得错误拒绝的概率近似或等于 α，此时就确定下了拒绝域

观测到数据 x1,…,xn 后，计算统计量 S 的值 s=h(x1,…,xn)，若 s 落入拒绝域，则拒绝假设 H0。

 

2.4 拟合优度检验

检验给定的分布列是否和观测数据保持一致。

广义似然比检验：

  1. 通过最大似然来估计模型，得到使似然函数最大的参数向量 θ^

  2. 进行似然比检验，即计算广义似然比：

    pX(x;θ^)pX(x;θ∗)

    其中 θ∗ 是原假设 H0 的模型。若广义似然比超过临界值 ξ，则拒绝假设。同样我们可以选择 ξ 以使得错误拒绝的概率近似或等于 α。


χ2 检验：设 θk∗ 是假设 H0 下随机变量取值 k 的概率，抽取样本量为 n 的样本，令 Nk 是样本中结果为 k 的次数，观测值为 nk。

  • 利用统计量：

    S=∑k=1mNkln⁡(Nknθk∗)

    或其二阶泰勒展开的近似量的两倍：

    T=∑k=1m(Nk−nθk∗)2nθk∗

    以及拒绝域：

    {2S>γ}或{T>γ}

    进行检验。

  • 临界值 γ 按照自由度为 m−1 的 χ2 分布的概率分布函数表确定，满足：

    P(2S>γ;H0)=α

    其中 α 是给定的显著性水平。
